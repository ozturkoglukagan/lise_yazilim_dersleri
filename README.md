Nihan ve Tuna ile Kodlamaya giriş
===
Bugün öğrendiğimiz kelimeler;

Bilgisayar Komutları

ls (macOS)& dir (windows)-> listelemek, bulunduğumuz konumdaki dosyaları gösterir
cd  -> change directory, bulunduğun konumu değiştirmek için
mkdir -> make directory, yeni bir dosya oluşturmak için
cd .. -> bulunduğun konumdan bir üst dosyaya çıkmak için

GitHUB Komutları

git clone -> oluşturulmuş bir repository (dosyayı) githubdan aynısından bir tane daha oluşturmaya yarar.

git add . -> dosya konumunda yapılan bütün değişiklikleri eklemeye yarar.

git commit -m "aciklama" -> dosya konumunda yapilan değişiklikleri açıklayan yazı yazılır.

git push -> bilgisayarımda yaptığım değişiklikleri githuba gönderebilirim.

git pull ->githubdaki değişiklikleri bilgisayarıma çekmek için kullanırım.
